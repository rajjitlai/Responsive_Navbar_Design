# Responsive Navbar with Search and Login

A modern, professional, and fully responsive navigation bar featuring integrated search and login modals. Built with HTML, SCSS/CSS, and Vanilla JavaScript.

![Project Preview](assets/preview.png)

## 🚀 Features

- **Dark Terminal Theme:** Sleek dark gradient background with high-contrast typography.
- **Glassmorphism Navbar:** Modern "glass" effect with background blur and subtle borders.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop views.
- **Search Modal:** Interactive search overlay for quick access.
- **Login Modal:** Sleek login interface for user authentication.

## 🛠️ Built With

- **HTML5:** Semantic structure.
- **SCSS/CSS:** Custom styling with CSS variables and responsive design.
- **JavaScript:** Interactive components for menu, search, and login toggles.
- **Remix Icon:** High-quality icon library.
